# 팀 랜덤 선택기 (Team Random Selector) - PRD

## 1. 개요 (Overview)
이 프로젝트는 팀원들이 회식 메뉴를 정하거나 회의 발표 순서 등을 정할 때 사용할 수 있는 무작위 추첨 웹 애플리케이션입니다.
모던하고 심플한 디자인과 함께, 추첨 시 부드럽고 가벼운 애니메이션을 통해 사용자 경험에 재미를 더합니다.
모바일에서도 편하게 사용할 수 있어야합니다.

## 2. 목표 (Goals)
*   **간편한 공유:** URL만으로 팀원들 목록(사다리/룰렛 데이터)을 바로 띄울 수 있어야 합니다.
*   **파일 지원:** 텍스트 파일(.txt)을 통해 다수의 팀원 이름이나 메뉴를 쉽게 입력받을 수 있어야 합니다.
*   **시각적 즐거움:** 모던하고 심플한 UI에 어울리는 부드러운 애니메이션 효과 적용.

## 3. 기술 스택 (Tech Stack)
*   **프레임워크:** Next.js (React 기반 핵심 로직 및 라우팅)
*   **스타일링:** Tailwind CSS (단순하고 모던한 UI 빠른 구성)
*   **애니메이션:** Framer Motion 또는 순수 CSS Transitions (부드러운 추첨 이펙트용)

## 4. 핵심 기능 (Key Features)

### 4.1. 데이터 입력 (Data Input)
1.  **URL 파라미터 (Query String):**
    *   예: `?members=김철수,이영희,박지성` 또는 `?menus=마라탕,돈까스,제육`
    *   URL에 포함된 데이터를 쉼표(`,`) 기준으로 파싱하여 화면상에 곧바로 인풋 값으로 채웁니다.
2.  **텍스트 파일 업로드 (File Upload):**
    *   `.txt` 파일을 드래그 앤 드롭 하거나 파일 선택 버튼을 통해 업로드합니다.
    *   각 줄(Line)이나 쉼표(Comma)를 기준으로 팀원/메뉴 데이터를 파싱합니다.

### 4.2. 무작위 추첨 모드 (Randomizer Modes)
1.  **단일 추첨 (Pick One):**
    *   입력된 목록 중 단 한 가지만 무작위로 뽑습니다 (예: 회식 메뉴 선택).
    *   애니메이션: 슬롯머신처럼 이름이 빠르게 지나가다 하나에 멈추거나, 카드들이 섞인 후 한 장이 뒤집히는 효과.
2.  **순서 정하기 (Shuffle/Ranking):**
    *   입력된 전체 목록의 순서를 무작위로 섞어 순위를 부여합니다 (예: 발표 순서).
    *   애니메이션: 리스트 아이템들이 서로 위치를 무작위로 교환하며 재배치되는(Reordering) 효과.

### 4.3. UI/UX 디자인 (Design & Aesthetics)
1.  **모던 & 심플:** 
    *   불필요한 색상을 배제하고, 여백을 넉넉히 주어 깔끔한 인상을 전달합니다.
    *   곡선 테두리(Rounded)와 부드러운 그림자(Shadow)를 활용한 카드 UI.
2.  **애니메이션 (Micro-interactions):**
    *   추첨 버튼 클릭 시 긴장감을 주는 약간의 딜레이(예: 1~2초간 섞이는 효과) 적용.
    *   결과 공개 시 파티클(폭죽) 이펙트 또는 강조되는 스케일 업 애니메이션.

## 5. 향후 일정 (Milestones)
1.  **초기 세팅:** Next.js & Tailwind CSS 프로젝트 구성
2.  **UI 뼈대 (Layouts):** 메인 렌더링 화면 및 파일/URL 파라미터 입력부 퍼블리싱
3.  **로직 구현:** URL/Text 데이터 파싱 및 무작위 랜덤 추첨 (랜더마이징) 기능 구현
4.  **애니메이션 & 고도화:** 슬롯머신/리스트 셔플 애니메이션 도입 및 배포 준비
